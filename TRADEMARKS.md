This project uses a neutral provider-independent product name: MessageBridge by
Bacaloo.

## Allowed Descriptive Use

Third-party names may be used in plain text when required to describe
compatibility, configuration, or documentation, for example:

- "adapter for the Meta WhatsApp Business Cloud API"
- "configure the provider phone number ID"
- "send a provider-approved template message"

Such use must be factual, limited, and non-promotional.

## Disallowed Use

Do not:

- name this project, a fork, package, domain, or app listing after a third-party
  brand unless explicitly authorized
- use third-party logos or icons as this project's logo or app icon
- imply partnership, sponsorship, certification, or endorsement
- combine this project's name or logo with a third-party logo
- use "official", "certified", "approved", or similar terms unless there is a
  written authorization that allows that exact statement

## Required Disclaimer

Public documentation that mentions a third-party provider should include a
clear disclaimer such as:

> MessageBridge by Bacaloo is an independent project and is not affiliated
> with, endorsed by, sponsored by, or operated by any messaging platform
> provider. Third-party names and trademarks are the property of their
> respective owners.
