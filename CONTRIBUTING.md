# Contributing

Contributions must follow the licensing and trademark rules in this repository.

Before adding provider-specific code or documentation:

- use official APIs and documented behavior where possible
- keep provider code in a provider adapter
- avoid third-party logos and brand assets
- avoid provider names in primary project identifiers
- document platform restrictions that affect behavior
- never commit credentials, access tokens, phone numbers, private webhook
  secrets, or production payloads
